# Program Authentikasi

Program interaktif Authentikasi yang dibuat menggunakan GO-Lang sebagai bahasa pemogramanya dan sebagai user interfacenya menggunakan package ```fmt.Scanf()```.

### Binnary Files: 
- Mac Os: [orion_k-darwin-arm64](https://github.com/bildanjhry/koda-b8-golang5/releases/download/v1.0.28/orion_k-darwin-arm64)
- Linux: [orion_k-linux-amd64](https://github.com/bildanjhry/koda-b8-golang5/releases/download/v1.0.28/orion_k-linux-amd64)
- Windows: [orion_k-windows.exe](https://github.com/bildanjhry/koda-b8-golang5/releases/download/v1.0.28/orion_k-windows.exe)

### Tech stacks:
- Go v1.26.4

### Layanan yang disediakan:
- Daftar/Register
- Login ke dashboard
- Forgot Password
